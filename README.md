# Multi-Branch Network Design and Security

## Overview

This project focuses on designing and securing a network that connects multiple branches in different locations. The goal was to build a reliable network and apply security measures to control and protect communication between departments.

## What I did

* Designed the network structure and topology
* Applied subnetting for different branches
* Configured VLANs to separate departments
* Implemented static and dynamic routing (OSPF)
* Used ACLs as a basic firewall
* Set up wireless security

## How it works

Each branch is connected through routers, and departments are separated using VLANs. Routing allows communication between branches, while ACLs and firewall rules control access to sensitive resources.

## Notes

This project helped me understand how real networks are designed and secured, and how different networking concepts work together.
